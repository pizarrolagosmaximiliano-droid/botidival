## Especificaciones técnicas:
- **Formato**: JPG o PNG
- **Tamaño recomendado**: 400x400 píxeles (mínimo 300x300)
- **Fondo**: Preferiblemente blanco o transparente
- **Calidad**: Alta resolución para buena visualización

## Instrucciones:
1. Toma fotos de alta calidad de cada producto
2. Recorta y edita las imágenes para que se vean profesionales
3. Guarda cada imagen con el nombre exacto especificado arriba
4. Coloca las imágenes en esta carpeta `images/`

Si no tienes una imagen para un producto específico, se mostrará el placeholder automáticamente.
