# 🚀 Guía de Despliegue Rápido - NETLIFY

## ¿Por qué Netlify?

✅ **Gratis** - Sitio completo sin costo
✅ **Rápido** - Lightning fast CDN global
✅ **Seguro** - HTTPS automático
✅ **Fácil** - Solo arrastrar y soltar
✅ **Profesional** - Dominio personalizado con descuento

---

## Paso 1: Crear una Cuenta Netlify (30 segundos)

1. Abre [netlify.com](https://netlify.com)
2. Haz clic en "Sign up" (arriba a la derecha)
3. Selecciona "GitHub" o "Email"
4. Completa el formulario
5. Verifica tu correo

✓ ¡Cuenta creada!

---

## Paso 2: Desplegar tu Sitio (2 minutos)

### Opción A: Arrastra y Suelta (Más Fácil)

1. En el dashboard de Netlify, busca la sección "Want to deploy a new site?"
2. Arrastra la carpeta `Boti Dival` a esa área
3. **Listo** - Tu sitio está en vivo! 🎉

**Tu URL será:** `nombre-aleatorio.netlify.app`

### Opción B: Usar GitHub (Recomendado a largo plazo)

1. Sube carpeta a GitHub
2. En Netlify: "New site from Git"
3. Conecta tu repositorio
4. Elige rama `main`
5. **Publicado automáticamente**

---

## Paso 3: Configurar tu Dominio Propio (OPCIONAL)

### Si Quieres Tener: botidival.com

#### Opción 1: Comprar dominio EN Netlify (Más fácil)

1. En Netlify → Site Settings → Domain Management
2. Haz clic en "Add custom domain"
3. Escribe: `botidival.com`
4. Haz clic en "Buy new domain"
5. Completa la compra (~$12 USD/año)
6. ✓ Listo, tu sitio está en botidival.com

#### Opción 2: Dominio de otro lugar

Si ya tienes un dominio (GoDaddy, Namecheap, etc.):

1. En Netlify → Domain Management → Add custom domain
2. Ingresa tu dominio
3. Copia los registros DNS de Netlify
4. Enlaza con tu proveedor de dominio
5. ✓ En 24-48 horas estará activo

---

## Paso 4: Verificar que Todo Funciona

Después de desplegar, verifica:

✅ Abre tu URL en el navegador
✅ Carga sin errores
✅ Los productos aparecen
✅ Búsqueda funciona
✅ Carrito es funcional
✅ WhatsApp se abre al presionar botones
✅ Se ve bien en móvil (abre DevTools F12 → Toggle device)

---

## Paso 5: Optimizaciones Posteriores

### 5.1 Agregar a Google Search Console

1. Abre [search.google.com/search-console](https://search.google.com/search-console)
2. Haz clic en "Add property"
3. Ingresa tu URL (botidival.netlify.app o botidival.com)
4. Verifica propiedad
5. **Resultado:** Tu sitio aparecerá en búsquedas de Google

### 5.2 Configurar SSL (HTTPS)

✓ Netlify lo hace automáticamente, sin hacer nada.

### 5.3 Activar Compresión Automática

En Netlify Site Settings → Build & Deploy:
- La compresión GZIP está habilitada por defecto
- Tu sitio carga ultra rápido

---

## Paso 6: Actualizar el Sitio Después

### Si usas GitHub:
```bash
git add .
git commit -m "Actualización de productos"
git push
```
✓ Se actualiza automáticamente

### Si usas Drag & Drop:
1. Descarga la carpeta actualizada
2. Arrastra de nuevo en Netlify
3. ✓ Actualizado al instante

## Troubleshooting Netlify

| Problema | Solución |
|----------|----------|
| "Dominio no se conecta" | Espera 24-48h si es nuevo dominio |
| "Sitio dice 404" | Asegúrate que index.html esté en la raíz |
| "Archivos no cargan" | Revisa que TODOS los archivos estén subidos |
| "Lentitud" | Purga caché: Site settings → Purge cache |
