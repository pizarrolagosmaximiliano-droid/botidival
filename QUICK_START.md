# ⚡ INICIO RÁPIDO - Boti Dival (3 PASOS)

## 🎯 Poner en vivo en 3 minutos

---

## PASO 1️⃣: Verifica los archivos

Asegúrate de tener estos 5 archivos:
```
✓ index.html
✓ styles.css
✓ script.js
✓ sw.js
✓ manifest.json
```

---

## PASO 2️⃣: Abre la web AHORA

### Opción A (Más fácil):
Haz clic derecho en `index.html` → "Abrir con" → Navegador

### Opción B (Servidor local):
```bash
cd "C:\Users\maxim\Desktop\Boti Dival"
python -m http.server 8000
```
Luego abre: `http://localhost:8000`

---

## PASO 3️⃣: Publica en Internet (GRATIS)

### 30 segundos con Netlify:

1. Abre [netlify.com](https://netlify.com)
2. Crea cuenta (email o GitHub)
3. Arrastra la carpeta `Boti Dival`
4. **¡LISTO!** 🎉 Tu URL aparecerá en segundos

**Resultado:** `algo.netlify.app` (URL completa)

## 🔧 PERSONALIZAR (5 min)

### Cambiar número WhatsApp
Busca en todos los archivos: `56985062378`
Reemplaza por: Tu número (ej: `56964044114`)

### Cambiar nombre de la botillería
En `index.html`:
- Busca: `Boti Dival`
- Reemplaza por: Tu nombre

### Cambiar horarios
En `index.html`, línea ~185:
```html
<div class="hour-row">
    <span>Domingo a Jueves</span>
    <strong>Hasta las 00:30</strong>  <!-- Cambiar aquí -->
</div>
```

### Cambiar colores
En `styles.css`, línea 5-6:
```css
--primary-color: #D4AF37;      /* Dorado - cambiar aquí */
--secondary-color: #C41E3A;    /* Rojo - cambiar aquí */
```
