## 🛠️ ARCHIVOS TÉCNICOS (No editar/guardar referencia)

### Archivos Principales de la Web

**[index.html](index.html)** 📄
- Estructura de toda la página
- 400+ líneas HTML5
- Incluye: header, hero, catálogo, carrito, footer
- Editar: textos, información, teléfonos

**[styles.css](styles.css)** 🎨
- Todos los estilos y diseño
- 700+ líneas CSS3
- Variables de color personalizables
- Responsive design incluido
- Editar: colores, fuentes, espacios

**[script.js](script.js)** ⚙️
- Toda la lógica y funcionalidad
- 400+ líneas JavaScript vanilla
- Sin dependencias externas
- Manejo del carrito, búsqueda, WhatsApp
- Editar: precios, categorías, productos

**[config.js](config.js)** ⚙️
- Configuración centralizada
- Datos del negocio
- Colores
- Horarios
- Editrar: información personal del negocio

### Archivos PWA (Progressive Web App)

**[manifest.json](manifest.json)** 📦
- Configuración de app móvil
- Íconos y colores
- Se puede instalar como app
- NO editar a menos que sepas de PWA

**[sw.js](sw.js)** 🔄
- Service Worker de caché
- Permite funcionamiento offline
- NO cambiar

### Archivos SEO y Servidor

**[sitemap.xml](sitemap.xml)** 🗺️
- Mapa del sitio para Google
- Helps indexación en buscadores
- Editar: reemplazar "tudominio.com" por tu URL

**[robots.txt](robots.txt)** 🤖
- Instrucciones para buscadores
- Editar: reemplazar "tudominio.com" por tu URL

**[.htaccess](.htaccess)** ⚙️
- Configuración de servidor Apache
- Caché, compresión GZIP, seguridad
- NO editar (solo si sabes de Apache)

### GitHub

**[.gitignore](.gitignore)** 📦
- Archivos a ignorar en git
- No tocar
